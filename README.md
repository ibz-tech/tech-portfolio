# Ibz-Tech Portfolio (GitHub Pages)

This repo hosts the **Ibz-Tech** portfolio website.

https://ibz-tech.github.io/portfolio/

It’s a **static site** (HTML + CSS) published using **GitHub Pages**.  
Right now it documents completed **EUS work experience at Schonfeld** and will be expanded over time.

---

## Files you will edit

- `index.html` — the page content (text/sections)
- `style.css` — the design (colors/layout)

---

## First-time setup (on Ibby’s machine)

### 1) Install the tools
1. **Git** (version control): https://git-scm.com/downloads  
2. **VS Code** (code editor): https://code.visualstudio.com/

### 2) Sign in to GitHub
- Make sure you can log in to: https://github.com/ibz-tech
